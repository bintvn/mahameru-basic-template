# MahameruJS Basic Template

This template is a part of [MahameruJS](https://www.github.com/bintvn/mahameru). To use this template, plase use [MahameruJS Project Initializer](https://www.npmjs.com/package/create-mahameru) or run this command bellow

```bash
npm create mahameru@latest
```

### Dependencies
* mahameru
* mysql2
* reflect-metadata
* typeorm

### Development Dependencies
* @types/node
* typescript


Rename .env.example to `.env.development` (for development) or .env.production (for production). Running npm run build triggers the Mahameru CLI to automatically copy `.env.production` into the `.mahameru` folder for production use or deployment.
