# MahameruJS Basic Template

This template is a part of [MahameruJS](https://www.github.com/bintvn/mahameru). To use this template, plase use [MahameruJS Project Initializer](https://www.npmjs.com/package/create-mahameru) or run this command bellow

```bash
npm create mahameru@latest
```

### Dependencies
* mahameru
* porterman

### Development Dependencies
* @types/node
* typescript
